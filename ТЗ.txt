Телеграм-бот для поиска фильмов (Kinopoisk API) с использованием библиотеки telebot

Техническое задание:

1. Общие требования:
-Проект разрабатывается на Python с использованием библиотеки telebot
-Должен быть портируемым (запуск через pip install -r requirements.txt)
-База данных - SQLite с использованием Peewee ORM

2. Команды бота:
-/start - начать работу с ботом
-/help - справка по командам
-/history - история поисковых запросов

3. Функционал поиска:
-Поиск по названию
-Ввод названия фильма/сериала
-Фильтрация по жанру
-Выбор количества результатов (1-10)
-Поиск по рейтингу
-Указание диапазона рейтинга (например, 7-9)
-Фильтрация по жанру
-Выбор количества результатов
-Поиск по бюджету
-Фильмы с низким бюджетом
-Фильмы с высоким бюджетом
-Выбор количества результатов

4. Требования к выводу информации

Для каждого фильма/сериала:
-Название
-Описание (первые 300 символов)
-Рейтинг KP
-Год выпуска
-Жанры
-Возрастной рейтинг
-Постер (если доступен)

Для истории запросов:
-Дата и время поиска
-Тип запроса
-Все данные о найденных фильмах
-Статус просмотра (просмотрен/не просмотрен)

5. Дополнительные требования:
-Пагинация при выводе результатов
-Возможность отмечать фильмы как просмотренные
-Фильтрация истории по датам
-Обработка ошибок пользовательского ввода
